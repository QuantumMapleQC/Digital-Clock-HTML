# Digital-Clock-HTML
# What is it used for ?
mainly used for my useless devices like my Samsung Galaxy Tab A (2017) SM-380 which runs always on charge and works with the HTML like a charm 24/7
![20251205_101204](https://github.com/user-attachments/assets/3a19d405-8d43-453a-9921-d719fbd26c36)

# What can it do ?
- Change background
- Custom background URL
- DEFAULT Fullscreens your device when entering the website
- Will refresh weather autonatically
- Asks your permission for Location access of your device for weather and timezone reasons

# Note that not every browser supports it
``Current Browsers Compatible``

  [x] Opera GX 
  
  [x] Google Chrome ( Older versions may not work)
  
  [ ] Firefox
  Other browsers are Unknown for now.

  # Issues / Errors that can happen 
  ``Geolocation error``: You may experience some technical difficulties by some location issues that you may have if that happens try manually finding your cordinates from your nearby city or town that may fix it! 
  
  ``Embed Fail``: make sure that your image / gif be properly embeded with a good link to make it work for the background to work. 
  
  ``Battery error``: Keep in mind that this website is in BETA which means its not gonna be as stable as you may expect so if you ever see that the current battery from the page is wrong compared to your device this is completely normal, (Future updates may fix it)
